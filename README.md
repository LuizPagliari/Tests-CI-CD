# Exemplo de Projeto Node.js com CI/CD

## Descrição
Projeto de exemplo para demonstração de pipeline de Integração Contínua e Deploy Contínuo utilizando GitHub Actions.

## Requisitos
- Node.js 16+
- npm

## Instalação
```bash
npm install
```

## Executando Testes
```bash
npm test
```

## Executando a Aplicação
```bash
npm start
```
